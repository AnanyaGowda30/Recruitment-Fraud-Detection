# Preemptive Fraud Detection and Risk Mitigation in Online Job Recruitment

## Major Project

### Technologies

- Python
- Flask
- DistilBERT
- XGBoost
- SHAP
- HTML
- CSS
- JavaScript

### Objective

Develop an AI-powered system capable of detecting fraudulent online job postings using Natural Language Processing and Machine Learning techniques.

### Features

- Fake Job Detection
- Recruiter Credibility Analysis
- Explainable AI (SHAP)
- Risk Score Prediction
- Interactive Dashboard
